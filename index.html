Measures of Central Tendency for Ungrouped Data

Mean:

Formula: X = ∑ given_number / total_number

X is the mean

Median:

Formula: Md = (total_number + 1) / 2

Md is the position and should always round up

Mode:

Unimodal - only one value that occurs with the greatest frequency

Bimodal - two values that are considered to be the mode

Multimodal - more than two values that are considered to be the mode

Geometric Mean:

Formula: X = ∑ given_number / total_number
